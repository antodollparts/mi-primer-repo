# Receta de Cheesecake — Página Web en HTML y CSS

Este proyecto es una página web estática que presenta una receta de cheesecake utilizando HTML y CSS. Su propósito es demostrar la estructura básica de una página web, organización de contenido y estilos visuales básicos.

## Instalación

1. Clona este repositorio:

   ```
   git clone https://github.com/antodollparts/mi-primer-repo.git
   ```

2. Navega al directorio del proyecto:

   ```
   cd mi-primer-repo
   ```

3. Ejecuta el proyecto:

   ```
   Abre el archivo index.html en tu navegador web
   ```
